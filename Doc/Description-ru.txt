Взгляните по новому на процесс тренировок с Gymme.

Приложение ориентировано на занятия силовыми видами спорта и бодибилдингом.  Вносите результаты в приложение между подходами - в следующий раз Gymme введет данные по тренировками, для того, что бы вы могли определить следующую цель. По вашим результатам Gymme соберёт статистику.

Приложение также следит за вашей активностью. Уже после второй тренировки Gymme начнет определять дату следующей и будет предлагать их вам по мере приоритета.

Испытайте новый подход к тренировкам в действии. Надеемся, Gymme понравится вам!
